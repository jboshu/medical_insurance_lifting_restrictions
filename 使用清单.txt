【快速开始 - 3个场景】

═══════════════════════════════════════
场景1：外网电脑准备离线包
═══════════════════════════════════════

1. 双击运行：prepare_offline.bat
2. 等待下载完成（生成 offline_packages 文件夹）
3. 将整个 Medical_insurance_lifting_restrictions 文件夹复制到U盘


═══════════════════════════════════════
场景2：内网电脑直接使用Python脚本
═══════════════════════════════════════

前提条件：
• 内网电脑已安装 Python 3.6+
• 已安装 Oracle Instant Client 并配置PATH

步骤：
0. 【重要】先检查Python环境：
   双击运行：check_python.bat
   
   如果检查失败，请先安装Python：
   • 下载：https://www.python.org/downloads/
   • 安装时勾选“Add Python to PATH”
   
1. 从U盘复制整个文件夹到内网电脑
2. 双击运行：install_offline.bat（安装依赖）
3. 如需修改数据库配置，编辑 oracle_db_tool.py
4. 双击运行：run.bat（启动程序）


═══════════════════════════════════════
场景3：内网电脑打包成独立EXE
═══════════════════════════════════════

前提条件：
• 内网电脑已安装 Python 3.6+
• 已安装 Oracle Instant Client 并配置PATH

步骤：
0. 【重要】先检查Python环境：
   双击运行：check_python.bat
   
1. 从U盘复制整个文件夹到内网电脑
2. 双击运行：install_offline.bat（安装依赖）
3. 双击运行：build_exe.bat（打包EXE）
4. 在 release 文件夹找到“医保限制解除工具.exe”
5. 将EXE文件和README.md复制到目标电脑
6. 目标电脑需安装 Oracle Instant Client


【重要提醒】

⚠️ Oracle Instant Client 是必需的！
   • 即使打包成EXE，仍需安装
   • 推荐版本：19.x 或 21.x
   • 必须添加到系统PATH环境变量
   • 下载地址：
     https://www.oracle.com/database/technologies/instant-client/downloads.html

【下一步操作】

根据你的情况选择：

A. 如果还在外网电脑：
   → 运行 prepare_offline.bat 下载离线包
   → 复制整个文件夹到U盘

B. 如果已到内网电脑且有Python：
   → 运行 install_offline.bat 安装依赖
   → 安装 Oracle Instant Client
   → 运行 run.bat 启动程序

C. 如果要打包成EXE：
   → 先完成B的所有步骤
   → 运行 build_exe.bat 打包
   → 分发 release 文件夹中的EXE
